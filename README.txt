Реализован шифр Виженера с использованием ЯП С++, 
позволяя пользователям шифровать и расшифровывать текст.

Установите и запустите на своем компьютере Visual Studio 
с рабочей нагрузкой Разработка классических приложений на C++.
Создайте шаблон проекта "Пустой проект".
С помощью обозревателя решений создайте элемент 
с расширением .срр внутри исходных файлов.
Скопируйте и добавьте текст программы (из Main.txt, тк в Sours.cpp не поддерживается кодировка), 
после чего запустите её с помощью локального отладчика.

При запуске программы пользователю дается выбор - на каком языке продолжить:
русский и английский.
Далее вводится текст, который можно:
- закодировать
- раскодировать.
Если пользователь выбрал вариант "закодировать", 
то ему необходимо ввести также и ключ, с помощью которого будет строиться шифрование.
На выходе получаем закодированный тест 
Если "раскодировать", то нужно ввести также и ключ, 
с помощью которого происходило шифрование.
На выходе получаем раскодированный тест
